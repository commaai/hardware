# comma hardware

This repo is the home for all comma.ai hardware.

Some of the hardware is fully open source and published here, and the rest is an archive of past hardware sold on the comma shop.

## comma shop

The [comma shop](https://comma.ai/shop) offers our latest hardware for sale.
